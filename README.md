What data does this tool need? It only needs basic true/false flags to track if the initial overlay was cleared and if the timer has finished.

Where is it stored? This data is stored locally within your browser's session storage.

Is it temporary or persistent? It is completely temporary and wipes clean the moment you close the tab.

Does the system need memory between sessions? No, it is designed to treat every new visit as a blank slate.

Does the system require AI inference? No, the tool relies entirely on basic, hardcoded JavaScript logic.

How many API calls are realistically required? Zero API calls are needed because the program runs entirely offline on your own machine.

What happens if the API fails? Since there is no external API to rely on, the tool cannot break from network outages.
